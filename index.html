look at pi: because numbers are pretty (and so are you)

What ... is this thing?

Well, I'm glad you asked. look-at-pi is a fun lil' toy that can visualize some irrational numbers. These aren't just random colors, I swear! Click on one of those symbols to the right to see. Then come back.

Then how ... does it work?

Magic!! Take Pi:
3.14159265...
Now divide it into 3 digit segments, like so:
314 | 159 | 265 | ...
We're doing well. Take modulus 256 of each chunk to cut it to that special 8-bit range. You'll get something like this:
058 | 159 | 009 | ...
Like wizards, we can now treat each set of 3 chunks as an (r,g,b) color value.
058159009 058159009
Now take Pi again, this time to quasi-infinity:

Instead of making one color from Pi, make a lot. And flash through 'em real fast. Suddenly each irrational number has a characteristic visual fingerprint. Whammy!
